Descripción del trabajo
Este repositorio contiene la resolución del Trabajo Práctico Nº 15, el cual consiste en la elaboración de consultas SQL sobre la base de datos *huertas_abp*, aplicando los conceptos de SELECT, JOIN, WHERE, GROUP BY y ORDER BY.
